dnsmasq 
=======
### Run all make commands from Makefile directory
`cd ~/.local/ubuntu/docker/dnsmasq`

### Build docker image
`make image`

### Make docker host configuration directory
`make config`

### Run docker container
`make container`

### Remove docker container
`make clean`

### Remove docker image
`make clean-image`

### Pull docker image
`make pull`

### Push docker image
`make push`
